# 🎄 Merry-Christmas 🎄

一個**超巨大**的終端機聖誕樹，讓你的命令列充滿節慶氛圍！✨

## 🌟 特色

- 🎨 **彩色閃爍燈泡**：隨機閃爍的彩色燈泡，讓你的聖誕樹活起來！
- ⏰ **即時倒數計時**：精準顯示距離聖誕節還有多久（天、時、分、秒）
- 🌲 **巨型尺寸**：60 行高、120 字元寬的超大聖誕樹
- ⭐ **閃亮星星**：樹頂有一顆閃亮的黃色星星
- 🎯 **自動更新**：畫面會持續更新，燈泡會不斷閃爍

## 🚀 快速開始

### 需求
- Python 3.6+
- 支援 ANSI 顏色碼的終端機（大部分現代終端機都支援）

### 安裝與執行

```bash
# 1. 下載專案
git clone https://github.com/your-username/Merry-Christmas.git
cd Merry-Christmas

# 2. 執行（不需要安裝任何套件！）
python src/app.py
```

就是這麼簡單！🎉

## 🎮 使用方式

1. 執行程式
2. 享受閃爍的聖誕樹和倒數計時
3. 按 `Ctrl+C` 停止動畫

## 🎁 技術細節

- **語言**：Python 3
- **依賴**：零依賴！只使用 Python 標準函式庫
- **顏色系統**：ANSI 256 色碼
- **動畫**：透過清屏和重新繪製實現

## 🎨 顏色說明

- 🟢 綠色：樹葉（▲）
- 🟡 黃色：星星（★）
- 🟤 棕色：樹幹（█）
- 🔴 紅色：標題文字
- 彩色燈泡（●）：隨機顯示紅、黃、藍、青、紫、橙、金、粉等顏色

## 💡 小提示

- 如果聖誕節已經過了，倒數計時會自動切換到下一年的聖誕節
- 在聖誕節當天，會顯示 "MERRY CHRISTMAS!" 訊息
- 建議使用全螢幕終端機以獲得最佳視覺效果
